# Урок 9 - Ansible Roles

### Задание

Написать Ansible роли, которые подготавливают сборочное и продовое окружение на 2-х нодах. На сборочной ноде производится сборка проекта, на продовой - запуск.

### Запуск

```bash
ansible-playbook main.yml
```
